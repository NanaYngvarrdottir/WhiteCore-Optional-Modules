This is the WhiteCore RegApi module. It (partly) implements the Registration Api
which allows Estate owners to create users that are bound to his Estates.

To Compile:

Copy the RegApi folder into the addon-modules directory and run prebuild and compile.

To Run:

Copy the Ini file from the Ini folder to the ServerConfiguration/Modules folder and start the WhiteCore.Server.exe file
When the RegAPI is loaded, it will open a new HTTP port and show the following in the console window

[RegAPI]: RegAPI has been started
